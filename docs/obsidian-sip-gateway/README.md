# SIP 实时语音网关 Obsidian 知识库

这是从项目学习笔记整理出来的 Obsidian vault。

打开方式：

1. 打开 Obsidian。
2. 选择 `Open folder as vault`。
3. 选择本目录：`docs/obsidian-sip-gateway`。
4. 从 [[00-index|00 总览]] 开始阅读。

说明：

- 仓库里的 `static/pages/notes.html` 仍是项目内静态学习页。
- 本目录用于 Obsidian 阅读、双链和长期知识整理。
- 文件名使用英文编号，正文标题使用中文，方便 Git 管理和 Obsidian 检索。
